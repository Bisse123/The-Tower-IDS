Welcome to The Tower ImportData!

This Google Apps Script tool helps you easily import and transfer game-related data for The Tower between different versions of your Google Sheets.

Get Started

Features

Easy Import/Export

Import data from one sheet to another with a single click.

Secure

Your data stays within your Google Account; no external servers are involved.

User-Friendly

Simple button interface—no technical expertise required.

Mobile User

Link to a web app is provided—no technical expertise required.

Flexible

Works with any The Tower-related Google Sheets you own or have access to.

How to Use

1

Open your game's Google Sheet

Start by opening the Google Sheet containing your The Tower game data.

2

Locate the "Import Data" button

Find the "Import Data" button in the IDS sheet.

3

Authorize the script

Authorize the script when prompted, so it can access your Google Sheets data.

4

Wait for completion

Wait for the import to complete. Your data will be transferred automatically!

5

Update sheet ID (Optional)

Optionally change your sheet ID in your IDS Master's Google Sheet to the new one.
